Definição das Disciplinas

Aplicações para Internet
É uma disciplina sobre o desenvolvimento web, com o estudo de HTMLL e CSS usados para Web Design.
Programação de Computadores
É uma disciplina que estuda o conceito de algoritmos e lógica de programação, utilizando as linguagens de programação Portugol e Python.
